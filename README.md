# Felipe Ontiveros Photography — Website

Personal website built for a family photography business.

🔗[felipeontiverosfotografia.netlify.app](https://felipeontiverosfotografia.netlify.app/)

## Built with:

- HTML5
- CSS3 (Flexbox, Grid, responsive design)
- JavaScript (image slider)
- Deployed on Netlify

## Features:

- Responsive layout
- Static image gallery
- Smooth scroll navigation
- Links to social media and WhatsApp contact
